# 页面缓存

keepalive是Vue的内置组件，作用是将组件缓存在内存当中，防止重复渲染DOM，属于消耗内存获取速度。常用的用法是将组件或者路由缓存。

警告

此配置只在 frameIn 中的路由有效。

# router配置缓存

对应路由配置meta属性：

{
  path: 'example',
  name: 'Example',
  component: () => import('@/viewxample/index.vue'),
  meta: {
    title: '示例',
    keepAlive: true,
  },
},
